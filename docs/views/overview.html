About

Welcome to the SOCIAL MEDIA NEWS FEED AGGREGATOR!! A cool portal which allows to integrate Twitter and Instagram accounts.

This is the front-end view for a social media aggregator, built on angular framework and bootstrap. It has been created to highlight the features of the middleware, which will be built during the course of the project.

The main features included on the front-end are new user sign up, existing user login, view of news feed collaboration, profile view and settings to edit your profile
