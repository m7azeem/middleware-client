About

Welcome to the SOCIAL MEDIA FEED AGGREGATOR!! A cool portal which allows to integrate Twitter and Instagram accounts.

This is the front-end view for a social media aggregator, built on angular framework and bootstrap. It has been created to highlight the fetures of the middleware, which will be built during the course of the project.

The main features included on the front-end are new use sign up, existing user login, news feed collaboration view, profile view and settings to edit the profile
